Exercise for Week 2 Power Coding Session

Important: Add a comment for each line of HTML you write!

  1. Create a copy of your template folder and paste it into your week2 folder.
  2. Rename this is folder to classexercise.
  3. In CSS: Select the body tag and set the margin to 0.
  4. In CSS: Select the ul tag and set the padding to 0.
  5. In CSS: Select the h1 tag and set font-size to 2em.
  6. In HTML: Create a nav tag with a class of navigation.
  7. In CSS: Add 4 css properties to the nav tag (background-color, margin, padding & text-align).
  8. In HTML: Create one ul tag inside the nav tag, and inside the ul: 3 li tags.
  9. In CSS: Add 4 css properties to the li tag (background-color, display, margin & padding).
  10. In HTML: Create one anchor tag inside each li tag.
  11. In CSS: Add 2 css properties to the a tag (color & text-decoration).
  12. In HTML: Below the navigation, create a section tag with a class of main.
  13. In CSS: Add 3 css properties to the section tag (background-color, text-align & display with a value of table).
  14. In HTML: Create a header tag inside the section tag.
  15. In HTML: Create one h1 tag inside the header tag with the content "My Favorite Super Heroes".
  16. In HTML: Create 3 div tags below the header, each with a class of box.
  17. In CSS: Add two css properties to those divs with a class of box (float to the left & width of 33.333333333%).
  18. In HTML: Inside each div add an img, h2 & p tag in that order.
  19. In HTML: Add a picture of one of your favorite superheroes to each img tag.
  20. In HTML: Add the name of the corresponding superhero inside each h2 as content.
  21. In HTML: Add a description of the corresponding superhero inside each p tag as content.
  22. In CSS: Select the img tag inside the div tags and add 2 properties (width of less than 100% & height).
  23. In CSS: Select the p tag inside the div tags and add 5 properties (background-color, border, height margin & padding).
