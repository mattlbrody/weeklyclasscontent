Project

Today we will start creating a project using the Google Books API.

Using the google books api, we will request data based on the user input (either title or author of a book) and display the data returned from the API, to the user.

We want to display the  author's name, the published date, an image of the book, and a link to learn more about the book. All of this info is included in the JSON data.

Steps For Making this JavaScript Project - Remember to break this project into manageable steps:

  1. Paper Prototype the layout of the project (draw out how you want the app to look) we will include each author's name, the published date, an image of each book, and a link to learn more about each book.
  2. Create the HTML needed to accept user data, a button to run the search function and a place to display the results.
  3. Style the project with flat design CSS, ideally something from codepen you modify to make your own.
  4. Write pseudocode for the JavaScript Logic (this is a halfway point between plain English and JavaScript).
  5. Write the JavaScript to capture user data, adding the answers to an array and displaying the results in the console.
