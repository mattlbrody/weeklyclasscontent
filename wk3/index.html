Project

This week we will focus on PSD to HTML/CSS Conversion. In Litmos Week 3 you'll find the Orange Juice image that we will recreate.

Steps For Completing a PSD Conversion

  1. Paper Prototype - Draw out what you want to create and choose what tags and properties you want to use.
  2. Try to recreate the HTML while adding comments to help you understand what the tags and attributes are doing. Reference w3schools and past projects for additional help.
  3. Try to recreate CSS while adding comments to help explain what the CSS properties or selectors are doing. Reference w3schools and past projects for additional help.
  4. Validate HTML and CSS code to make sure there are no errors.
  5. Upload your completed project to Github following the steps in "How to Post a New Project to Github using Git".
  6. Check off in Litmos in Week 3 in class project complete.
