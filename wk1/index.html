Exercises

Important: Make sure to add comments to your code.

  1. Create a copy of your template folder and rename it to week1exercise.
  2. Drag this new folder into your 'week1' folder inside your 'codify' folder.
  3. In HTML: Create a ul element with a class of nav.
  4. In CSS: Add three CSS properties to the ul (background-color, margin & padding).
  5. In HTML: Create three li elements inside the ul.
  6. In CSS: Add three css properties to the li (background-color, display & padding).
  7. In HTML: Create an anchor element to connect each li to one of your favorite websites.
  8. In HTML: Create 2 div elements below the navigation, each with a class of box.
  9. In CSS: Add two CSS properties to the divs (float to the left & width of 50%).
  10. In HTML: Inside 1 of the divs, create 6 header elements each with one of your favorite things you learned this week.
  11. In HTML: Inside the other div, create an image element with something you love as the picture.
  12. In CSS: Add three properties of the box model to the image.
  13. In HTML: Below the image, add a paragraph with some content (placeholder text is fine).
  14. In HTML: Add a second class to one of your divs.
  15. In CSS: Select the p element inside that div and add 3 properties (background-color, border, margin & padding).

Once you have completed these exercises, try adding some more divs to the page and experimenting with the following CSS properties.

  1. color, background-color
  2. elementary selectors: element, .class, #id & descendant selector: parent child
  3. box model elements: margin, border, padding, contents
  4. display: inline, block, inline-block
  5. border-radius, height & width
